# مائة سؤال عن الإسلام — 100 Questions About Islam

Interactive trilingual web app based on **"Mi'at Su'al 'an al-Islam"** (100 Questions About Islam) by **Sheikh Mohammed al-Ghazali** (1917-1996).

## About the Book

This book is a deep intellectual dialogue between the Egyptian thinker **Khalid Muhammad Khalid** (1920-1996) and **Sheikh Mohammed al-Ghazali**. Khalid asks bold, direct questions about Islam — from the simplest basics to the most complex contemporary issues. Al-Ghazali answers with an open mind and solid knowledge, presenting the true image of Islam.

## Features

- **Trilingual**: Arabic, English, French
- **25 Q&A cards** organized by 5 themes: Basics, Worship, Society, Misconceptions, Contemporary Issues
- **Smart search**: Type your question, get the closest answer from the book
- **Quiz**: "Who Wants to Be a Scholar?" — 15 questions with lifelines (50/50, Ask the Sheikh, Guidance)
- **Gamification**: XP points, 5 badges, daily streak tracking
- **Age modes**: Young Explorer (7-12) and Teen Scholar (13+)
- **3 themes**: Forest, Night, Oasis
- **Narrator**: Full audiobook mode with karaoke, speed/pitch control, sleep timer, duo reading
- **PWA**: Works offline, installable
- **Responsive**: Mobile-first design
